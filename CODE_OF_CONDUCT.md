# 行为准则

本项目希望为所有参与者提供一个专业、尊重、可协作的交流环境。

## 期望行为

- 尊重不同背景和经验水平的参与者
- 针对问题、代码和设计进行建设性讨论
- 在反馈中保持清晰、克制和专业
- 优先讨论事实、复现方式和可执行建议

## 不可接受的行为

- 人身攻击、侮辱、骚扰或威胁
- 恶意挑衅、歧视性言论或故意制造对立
- 持续发布与项目无关的垃圾信息
- 未经允许公开他人的敏感信息

## 适用范围

本准则适用于以下场景：

- Issues
- Pull Requests
- Discussions
- 与项目协作直接相关的其他公开交流渠道

## 处理方式

维护者可根据情况采取以下措施：

- 提醒并要求调整沟通方式
- 隐藏、编辑或删除不当内容
- 暂时或永久限制参与权限

## 联系维护者

如果你在项目协作中遇到严重不当行为，可联系仓库维护者处理。
